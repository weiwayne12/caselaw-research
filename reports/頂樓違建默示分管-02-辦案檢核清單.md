# 頂樓違建默示分管 — 辦案檢核清單

> 依臺灣高等法院民事判決整理，用於審理「其他住戶請求頂樓住戶拆除屋頂平台增建物、返還共有部分，並涉及默示分管契約抗辯」案件。

---

## A. 基礎權利與標的

- [ ] 系爭屋頂、頂樓平台、樓梯間上方平台是否屬區分所有建物共有部分？
- [ ] 建物興建、取得建照、使用執照時間為何？是否在公寓大廈管理條例施行前？
- [ ] 原告是否為區分所有權人或共有物共有人？
- [ ] 被告是否為頂樓房屋所有人、增建物所有人或事實上處分權人？
- [ ] 請求標的是否具體特定：拆除範圍、返還範圍、面積、附圖、測量成果圖？

## B. 默示分管契約成立

- [ ] 買賣契約、預售契約、使用權證明書、規約、管理辦法是否載有屋頂歸頂樓住戶使用？
- [ ] 是否有建商或起造人交付頂樓平台鑰匙、使用權證明、管理權說明？
- [ ] 增建物何時存在？有無航空照片、房屋稅籍、建管查報、照片、證人可證？
- [ ] 其他住戶何時入住？增建時是否已全體入住？
- [ ] 增建施工時，其他住戶是否知悉、反對或同意？
- [ ] 是否有長期依頂樓專用狀態收取較高管理費、清潔費或其他費用？
- [ ] 後手買受時，是否由外觀、現況說明書、買賣契約或管理費紀錄可得而知分管狀態？
- [ ] 使用型態是否排他？或仍容許其他住戶進出、曬衣、使用水塔、逃生？

## C. 分管契約抗辯的限制

- [ ] 分管契約內容是否明確包含「可加蓋建物」？或僅是「管理使用」？
- [ ] 增建物是否超出原來分管範圍？
- [ ] 後手是否改建、隔間、出租、加裝水塔、擴大占用？
- [ ] 其他住戶是否於後手改建後立即反對、檢舉或起訴？
- [ ] 是否仍保留通往屋頂平台、水塔、消防避難、救援所需之通道及空間？

## D. 違建與共有物通常用途

- [ ] 增建物是否經主管機關查報為違章建築？
- [ ] 單純違建以外，是否另有私法上妨害？
- [ ] 是否妨害屋頂平台作為避難場、屋頂出入口、共同天線、避雷針、水塔維修、消防通路等通常用途？
- [ ] 是否占滿或大幅占用屋頂平台？
- [ ] 是否作住宅、房間、出租使用，而非單純防漏、防曬或管理使用？
- [ ] 是否設門、上鎖、阻斷其他住戶使用？

## E. 結構安全與鑑定

- [ ] 是否主張增建物增加建物負重、影響耐震或結構安全？
- [ ] 是否有土木技師、結構技師、建築師公會鑑定？
- [ ] 鑑定是否具體指出增加重量、地震力、安全係數降低、樑柱受損或坍塌風險？
- [ ] 被告是否提出原先曾經建築師鑑定或主管機關備查？該鑑定面積是否與現況一致？
- [ ] 是否有現況面積擴大，導致舊鑑定不能作為安全依據？

## F. 請求權與裁判結果

- [ ] 民法第 767 條第 1 項前段：是否請求返還共有物？
- [ ] 民法第 767 條第 1 項中段：是否請求除去妨害，即拆除增建物？
- [ ] 民法第 821 條：是否為全體共有人利益而請求？
- [ ] 民法第 179 條：是否請求相當於租金之不當得利？
- [ ] 若認有分管契約，是否仍可准「拆除增建物」但駁回「返還平台／不當得利」？
- [ ] 若不認分管契約，是否准拆除、返還及不當得利？

## G. 主文與執行可能性

- [ ] 主文拆除範圍是否對應附圖、編號、面積？
- [ ] 若請求返還共有部分，主文是否表明返還予原告及其他全體共有人？
- [ ] 不當得利起算日、終期及按月給付是否明確？
- [ ] 是否需假執行及供擔保金額？
