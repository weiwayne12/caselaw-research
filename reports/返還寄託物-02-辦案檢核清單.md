# 返還寄託物 — 辦案檢核清單／審理流程 SOP

> 從 74 篇民事判決反覆出現之調查事項與程序節點歸納而成之**可勾選檢核表**。
> 用途為提示「該查什麼、該命兩造表明什麼」，**非裁判結論、非法律意見**；個案仍依具體事證認定。

---

## A. 程序面（受理～言詞辯論前）

- [ ] **管轄**：有無合意管轄條款（寄託單／使用條款常見）？涉外案件（虛擬資產平台、海外帳戶）有無國際管轄與準據法問題（涉外民事法律適用法 §20）？ → [34][60][72] 幣寶系列、[68] ACE
- [ ] **當事人能力／適格**：受寄人死亡之繼承人承受、法人清算人、遺囑執行人、遺產管理人之當事人適格？ → [5][37][53][54][64]
- [ ] **訴之變更追加**：原告是否於二審追加請求權基礎或聲明？是否「基礎事實同一」（民訴 §255／§446）？ → [13][20][28][31][44][52]
- [ ] **一造辯論**：被告（常為平台、失聯受寄人）受合法通知未到，是否符合民訴 §386？ → [42][47][67][68][69][70][73]
- [ ] **訴訟標的價額／裁判費**：金錢寄託以請求金額核定。

## B. 契約成立要件（爭點一，最須先釐清）

- [ ] **寄託物之交付**有何證據？（匯款明細、保管條／保管單、存證信函、對造他案自認、證人）
- [ ] **保管合意**有何證據？是否區別於「贈與／借貸／擔保／買賣價金／投資款」？
- [ ] 命**原告具體表明**：交付時間、方式、金額／標的、有無約定返還期限、有無約定利息。
- [ ] 命**被告具體表明**：否認交付？或承認收受但抗辯款項性質為何（借貸／贈與／擔保／已用於特定用途）？
- [ ] **親屬／伴侶間**交付：有無排除「親屬間金錢往來原因多端」之合理說明？

## C. 契約定性（爭點二）

- [ ] 標的為**金錢／代替物**（→ 消費寄託 §602／§603 準用 §478）或**特定物**（→ 一般寄託 §589／§597／§598）？
- [ ] 若為消費寄託：返還客體為「同種類、品質、數量」之物，主文措辭是否相應？
- [ ] 虛擬資產：返還標的之**幣種與數量**如何特定（主文可否執行）？ → [68] 命返還顆數、[73] 命返還本金＋利息

## D. 返還請求與催告（爭點三）

- [ ] 有無約定返還期限？未定期限者，原告是否已**定一個月以上相當期限催告**（§478 後段準用）？
- [ ] 催告之方式與到達（存證信函／律師函／起訴狀繕本送達）？
- [ ] **利息起算日**之認定（催告期滿翌日／起訴狀送達翌日）。

## E. 受寄人抗辯之調查（爭點四）

- [ ] **已返還（清償）**抗辯 → 提示由**被告負舉證責任**；查交付返還之證據。 → [19][26]
- [ ] 消滅時效、抵銷、款項已依約定用途支出等抗辯之事證。
- [ ] 一部返還之金額認定。

## F. 特別類型專屬節點

- [ ] **金融機構冒領（C 型）**：印鑑變更／補發存摺金融卡之核對程序、銀行有無盡善良管理人注意義務、冒領人是否為「債權準占有人」（持有存摺＋印章＋金融卡 v. 僅持竊得證件）。 → [9]
- [ ] **第三人異議之訴（D 型）**：執行標的為「存款債權」者，第三人對之僅有債權而無物權 → 提示第三人異議之訴須有所有權等物權（強執 §15）；借名存款人是否誤認有所有權。 → [2][35][43]
- [ ] **遺產／繼承（F 型）**：寄託物是否屬遺產、應返還予全體繼承人公同共有或按應繼分；遺囑執行人之權限。 → [37][53][64]
- [ ] **特定物返還不能（E 型）**：是否併為「如不能返還，給付相當價額」之代償給付聲明（民 §226）？ → [44][50]
- [ ] **虛擬資產平台停業（B 型）**：平台公告停業、主管機關公告、使用條款終止之認定。 → [69][70]

## G. 主文與假執行

- [ ] 金錢給付 v. 特定物返還 v. 代替物返還，主文措辭是否精確、可執行。
- [ ] 假執行擔保金之酌定。
- [ ] 訴訟費用分擔（一部勝訴之比例）。

---

### 常見「卡關／應再調查」訊號（語料中反覆出現）
1. 只證明「有交付金錢」卻未證明「保管合意」→ 爭點一舉證不足之最大宗。
2. 親屬間交付未排除贈與／扶養之可能。
3. 借名存款人誤以第三人異議之訴排除執行（應釐清其僅有債權）。
4. 未定期限消費寄託未先合法催告即起訴，影響利息及請求時點。
5. 特定物可能已滅失，未預為代償給付之聲明。
